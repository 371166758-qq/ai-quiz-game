# AI答题大师游戏

## 游戏简介

AI答题大师是一个基于Web的答题游戏，玩家需要回答关于人工智能的各种问题，测试和提高AI知识水平。

## 技术栈

- **游戏引擎**: Phaser.js 3.60
- **前端**: HTML5 + JavaScript
- **无需后端**: 纯前端游戏

## 如何运行

### 方法1: 直接打开
1. 双击 `index.html` 文件
2. 在浏览器中打开即可游玩

### 方法2: 本地服务器
```bash
# 使用Python启动简单服务器
cd ai-quiz-game
python3 -m http.server 8080

# 然后在浏览器打开
# http://localhost:8080
```

### 方法3: 部署到线上
- 上传到GitHub Pages
- 部署到Vercel
- 部署到Netlify
- 或任何静态网站托管服务

## 游戏特色

### 核心玩法
- 10道AI相关题目
- 每题10分，满分100分
- 答对继续，答错结束
- 30秒答题时间

### 题目类型
- 选择题（单选）
- 判断题（对错）
- 详细答案解析

### 题目分类
- AI基础知识 (40%)
- AI工具使用 (30%)
- AI应用场景 (20%)
- AI前沿动态 (10%)
